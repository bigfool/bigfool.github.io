看日志发现有个严重错误,Qt库不兼容

3月 20 20:05:07 [Genymotion] [Fatal] Cannot mix incompatible Qt library (version 0x40806) with this library (version 0x40802)

我是安装过Qt相关的库,那么可以不用genymotion自带的Qt库. 先安装libxi-dev libxmu-dev两个库:

sudo apt-get install libxi-dev libxmu-dev

再将genymotion文件下的Qt库移走:

sudo mkdir QtLibs sudo mv Qt.so* QtLibs/

这样就解决了问题。
